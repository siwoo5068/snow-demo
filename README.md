# 🌨️ Snow Crawler: Prologue

**혹한의 눈보라 속, 제한 시간 안에 가구를 파밍하고 오두막으로 돌아오는 1인칭 텐션 생존 게임**

## 🎮 게임 개요

- **장르:** 1인칭 3D 텐션 파밍 & 방 꾸미기 생존
- **엔진:** Unity 6 (6000.3.11f1)
- **상태:** 프로토타입 (개발 중)

### 핵심 경험

> 눈보라 속에서 제한 시간 내에 아이템을 파밍하고 안전한 오두막으로 귀환하세요.
> 무거운 가구를 챙길수록 이동 속도가 느려집니다 — **무엇을 가져가고, 무엇을 버릴 것인가?**

## 🕹️ 조작법

| 키 | 동작 |
|----|------|
| `W/A/S/D` | 이동 |
| `마우스` | 시점 회전 |
| `E` | 상호작용 (아이템 줍기 / 제작대 사용) |

## 🧩 구현된 시스템

### 코어 생존 및 다층 엔딩
- **점진적 체온 변화:** 안전지대(오두막) 진입 시 한 번에 회복되던 체온을 `RecoverySpeed`에 따라 서서히 차오르게 하여 긴장감과 안도감을 교차
- **재료 파밍 & 장비 제작:** `SupplyBox`를 파밍해 Material을 모아 `CraftingTable`에서 코트를 다단계로 업그레이드
- **복합 승리 조건:** 기존 방식이 아닌 "코트 레벨 2 이상 + 오두막 내 가구 안락도 60% 이상 복구"를 달성해야 무전기(Radio)로 엔딩 가능

### 무게 딜레마
- 가구마다 고유 무게 보유 (0.5kg ~ ??kg)
- 무게 증가 → 이동 속도 감소 (최저 20%까지)
- 무게 HUD로 현재 적재량 시각적 확인 (위험도에 따라 색상 변화)

### 체감 피드백
- **발소리:** 무거울수록 간격 넓어짐 + 피치 낮아짐 + 볼륨 커짐
- **카메라 흔들림:** 무거울수록 걸을 때 상하 진폭 커짐

### 분위기 연출
- **얼어가는 시야:** 시간이 줄수록 화면이 푸른 서리로 덮이는 오버레이 (지수 곡선)
- **귀환 시 온기:** 별장 진입 시 조명이 차가운 파란색 → 따뜻한 주황색으로 전환
- **크로스페이드 오디오:** 실내외 전환 시 눈보라 효과음과 오두막 BGM이 끊기지 않고 자연스럽게 페이드 처리(`ZoneAudio`)

### 몰입형 인터페이스 (UX/UI)
- **오두막 나침반(CabinCompass):** 인디 게임의 텐션을 해치지 않도록, 화면 중앙이 아닌 가장자리의 작은 점이 오두막 방향을 안내
- **단일 상호작용 시스템:** 중앙 크로스헤어 조준점에 위치한 타겟에 맞추어 "E: 코트 업그레이드" 등 상황별 가이드가 자동 팝업

## 📁 프로젝트 구조

```
Assets/Scripts/
├── AtmosphereManager.cs    # 서리 오버레이 + 조명 전환 연출
├── CabinCompass.cs         # 화면 가장자리 나침반 방향 안내 UI
├── CameraBob.cs            # 무게 기반 카메라 흔들림
├── FootstepSound.cs        # 무게 연동 발소리(간격/가속/볼륨)
├── GameProgress.cs         # 안락도 점수 및 무전기 승리 조건 관리
├── Interaction.cs          # 가구, 작업대, 재료 등 모든 상호작용 물리 처리 통합
├── InteractionPrompt.cs    # 상황별 조준점 동적 팝업 가이드 UI
├── ItemDatabase.cs         # 파밍 오브젝트 및 무거운 가구 종류 + 무게 수치 제공
├── PlayerController.cs     # 1인칭 이동 + 가구 무게 비례 속도 감소 조율 
├── SurvivalTimer.cs        # 점진적 체온 변화 및 한파 시간, 코트 업그레이드 연동
├── WeightHUD.cs            # 무게 단계별 색상 경고 UI
└── ZoneAudio.cs            # 눈보라와 실내 환경음 크로스페이드
```

## 🔧 설계 원칙

- **역할 분리:** 각 스크립트는 하나의 역할만 수행. 강한 결합(Coupling)을 지양.
- **읽기 전용 연동:** 새 시스템은 기존 스크립트의 public 값을 읽기만 할 뿐, 내부 로직을 침범하지 않음.
- **확장 가능 구조:** 서리 이미지, 사운드 에셋 등을 Inspector에서 교체하면 코드 수정 없이 비주얼 업그레이드 가능.
